## Notes de conception sur ce format

- **Code couleur par type** (🟥 reco, 🟦 PNDS, 🟩 essai, 🟨 alerte) : balayage
  visuel rapide.
- **La section « ce qui change » est toujours datée** par rapport à une
  version antérieure explicite — jamais un diff vague.
- **Les limites d'accès sont affichées** (« synthèse fondée sur l'abstract »).
- Longueur cible : 4–8 items synthétisés + liste brute ; ≤ 15 min de lecture.
- L'email reprend ce contenu ; l'archive web ajoute les tags par pathologie.
